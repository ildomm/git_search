## 🚀 Getting Started

### Prerequisites

- Ruby 2.6+
- Make

### Quick Setup

```bash
# Install and setup everything
make setup

# Start the server
make server

# Open in browser
make open
```

## 💻 Development

```bash
# Show all available commands
make help

# Common tasks
make test    # Run test suite
make lint    # Check code style
```
